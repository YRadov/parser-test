# Usage

### Clone reposytory
```
$ git clone git@github.com:YRadov/parser-test.git
```
### Go into the created folder
```
$ cd parser-test/
```
### Install all dependencies
```
$ composer install
```
### Start unit tests
```
$ ./vendor/bin/phpunit --bootstrap vendor/autoload.php tests/DateParserTest
```
### Data for tests are in the 
`/tests/dates.php`
